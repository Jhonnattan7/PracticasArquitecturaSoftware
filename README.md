# Prácticas de Arquitectura de Software

Este repositorio contiene las prácticas y laboratorios del curso de Arquitectura de Software.

## Estructura del Repositorio

### Laboratorio 1 - Python
- Implementación de sistema de tutorías usando arquitectura hexagonal
- Incluye API REST con FastAPI
- Pruebas unitarias

### Laboratorio 2 - UML y OCL
- Diagramas UML (casos de uso, secuencia, dominio)
- Restricciones OCL usando USE
- Implementación en Python de dominio con restricciones
